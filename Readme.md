### DEPLOY ON TERMUX
1. 
```sh
$ pkg upgrade -y && pkg update -y
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone your forked repo.git
$ cd alpha-md
$ npm i 
$ npm start
```
